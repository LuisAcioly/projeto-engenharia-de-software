# IBurguer

O sistema iBurguer desempenha as funções de um aplicativo de delivery de hamburgueres, permitindo aos cliente pedirem seus 
hamburgueres pelo celular ou browser de computador.

## Usuarios:

- Cliente
- Admin

## Entidades:
- Cliente
- Hamburguer
- Bebida
- Pedido

## Principais funções:

- Cadastrar cliente
- Solicitar pedido
- Cadastar hamburguer
- Alterar hamburguer
- Excluir hamburguer
- Excluir conta
- Processar pedido
- Realizar login
- Visualiar perfil
- Alterar perfil
- Visualizar pedidos
- Cancelar pedido 

## Tecnologias utilizadas:

HTML5, CSS3, PHP 7, Javascript e Adobe Photoshop CC 2019
