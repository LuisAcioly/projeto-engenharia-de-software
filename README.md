# IBurguer

O sistema iBurguer desempenha as funções de um aplicativo de delivery de hamburgueres, permitindo aos cliente pedirem seus 
hamburgueres pelo celular ou browser de computador.

## Usuarios:

- TesteCliente
- Admin

## Entidades:
- Cliente
- Hamburguer
- Bebida
- Pedido

## Principais funções:

- Cadastrar cliente
- Realizar pedido
- Cadastar hamburguer
- Alterar hamburguer
- Excluir hamburguer
- Excluir conta
- Processar pedido
- Realizar login
- Visualiar perfil
- Alterar perfil
- Visualizar pedidos
- Cadastrar Bebida
- Excluir bebida 

## Tecnologias utilizadas:

HTML5, CSS3, PHP 7, Laravel e Adobe Photoshop CC 2019
